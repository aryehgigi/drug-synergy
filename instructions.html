Goal:

In this task, you will need to find interactions and classify them. There are three types of classes to classify: SYNergestic, INTerating (cases that we are sure that there is an interaction but it is unclear whether or not it's synergistic), and lastly NEGative interactinos. As summerized in the following table:
Interaction description Tagging label What to annotate Example
Synergistic interaction SYN all
Interaction exists but unclear if synergistic INT all
Not interacting - Don't annotate
Interacting in a Negative mennar NEG all

Task guidlines:

  1. Only for the INT cases, you should click on the "Get Full Context" button and use it to detrmine the type of interaction, if possible (otherwise it should indeed be marked as INT).
  2. An interaction can occur between more than two drugs, if so notice that we don't need each pair from this group to have a marked interaction, as long as they all connect to the same graph. e.g. "Drugs A, B and C are synergistic." connecting A to B and B to C is sufficient, no need to connect A to C.
  3. In case you reach the conclusion that the interaction is synergistic, only using the context, please check the button stating that the context is required (for the specified relation).
  4. If the sentence contained temporal information check the relevant button (for the specified relation).
  5. A warning would pop up if you didn't fill up the answers for the context and temporal quesions (for the relevant labels).
  6. The words that are marked as drugs may be partial. Feel free to add more drugs that are not marked in the sentence.
  7. Each interaction should be marked with a different tag (SYN1, SYN2... for synergestic interactions, and NEG1, NEG2... for negative interactions, and INT1, INT2.. for unclear interactions). We beleive that tree of each should be sufficient for a given sentence.

Technical guidlines:

  1. To mark an interaction be sure to be on the Assign-Relations mode/button, them choose the relevant label (e.g. SYN1), and press on the participating drugs.
  2. In case you want to use a span of words as a drug, you can merge the words together by pressing the Select Spans button and dragging the mouse to catch the target words.
  3. Your annotations are saved once every 5 examples. So if you want to undo changes dont wait to long. Alternatively if you want to save and leave you can press Ctrl+s or click on the save button.
  4. When done with an annotation, click the big-green-accept button, then you would be fed with the next annotation.
  5. In case you wish to undo your last accepted or skipped annotation, you can press the big Undo button. In case you want to skip a sentence due to a problem with that setnence simply press on the big skip/ignore button.
